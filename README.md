# DevOps & Infrastructure Portfolio

Bu depo, gerçek dünya senaryolarında uyguladığım altyapı kurulumlarını ve sistem yapılandırmalarını içerir.

## 🚀 Proje İçeriği

* **`docker-compose.yml`**: Mikroservis mimarisi (Backend, DB, Redis) için konteyner orkestrasyonu.
* **`wg0.conf` & `clients/`**: Güvenli uzaktan erişim ve Site-to-Site VPN için WireGuard yapılandırmaları.
* **`backend_env_yedek.txt`**: Staging/Production ortamları için environment değişken yönetimi.
* **`Nginx Configs`**: Reverse Proxy, SSL sonlandırma ve yük dengeleme ayarları.

## 🛠 Teknik Yetkinlikler
- **Containerization**: Docker, Docker Compose
- **Network & Security**: VPN (WireGuard), Firewall Management
- **CI/CD**: Jenkins, Deployment Pipeline Management
- **Web Servers**: Nginx
